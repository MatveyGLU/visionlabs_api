# visionlabs_api
типичное окно регистрации пользователя в некотором сервисе, например, онлайн-курсов или соц.сети. Пусть эта регистрация осуществляется наиболее классическим образом по двум параметрам: email и password.  Задача: поставить задачу на backend разработку этого метода регистрации по URL v1/user, в которой будет достаточное описание всех вводных.
